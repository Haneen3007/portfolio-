My personal portfolio website, showcasing my work as a **Full-Stack Web Developer** and **UI/UX Designer**.

**🔗 Live site:** [haneen3007.github.io](https://haneen3007.github.io)

## About

I'm a Software Engineering student (SIM) at the Faculty of Science, Alexandria University, specializing in:

- **Frontend Development** — HTML5, CSS3, JavaScript, responsive design
- **Backend Development** — PHP, MySQL, RESTful APIs, authentication systems
- **UI/UX Design** — Figma wireframing & prototyping
- **Networking & Security** — CCNA, SOC Fundamentals, CEH

## Features

- ⚡ Single-file website — pure HTML, CSS & JavaScript, no frameworks, no build step
- 📱 Fully responsive — works on mobile, tablet, and desktop
- 🎨 Custom design system — "web stack layers" concept reflecting my full-stack skills
- ✉️ Working contact form with spam protection
- ♿ Accessible — keyboard focus states and reduced-motion support
- 🚀 Fast — optimized embedded assets, zero external dependencies besides fonts

## Tech Stack

| Layer | Technology |
|---|---|
| Structure | Semantic HTML5 |
| Styling | Modern CSS (Grid, Flexbox, custom properties) |
| Interactivity | Vanilla JavaScript (typing animation, scroll reveal) |
| Contact form | FormSubmit |

## Run Locally

cd portfolio
# open index.html in your browser
